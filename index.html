This mind map outlines the process of converting Neuropixel data into 3D visualizations of neural activity, divided into three key areas: Background, which explores intra-cortex communication, decision-making, and neural activity mapping using experimental and deep learning approaches; Data Analysis, which details methods like noise reduction, spike sorting, regression models, and statistical validation to extract accurate neural signals; and 3D Visualization of Spikes Location, which focuses on mapping neural spike data onto the Allen Mouse Brain Atlas using tools like Brainrender, Blender, and Python to create high-resolution 3D visual representations.

Abstract

The Imperial Pixelminds project focuses on transforming Neuropixel data from 2D to 3D visualizations of neuronal activity in the mouse brain. Neuropixels are advanced probes that capture electrical signals from neurons and are typically visualized using the Allen Mouse Brain Common Coordinate Framework (CCF) in a 2D format. This project aims to overcome the limitations of 2D visualizations by developing a 3D Brainrender model to improve image clarity and spatial precision.

The project is divided into four steps: (1) Spike Categorization: Applying vision, action, and choice kernels with half-Gaussian filters to classify neuronal spikes. (2) 3D Spike Localization: Using the Adam optimization algorithm to map spike coordinates from Neuropixel channel data. (3) Spike Allocation in a 3D Brain Model: Plotting 1,000 spike locations onto a 3D mouse brain model. (4) Spike Functionality Visualization: Assigning each spike a functional category and creating a chromatic 3D brain image.

The team identifies technical, project management, and resource risks, emphasizing robust documentation and effective communication. Their ultimate goal is to advance neural mapping, enhance data interpretation, and contribute to a new 3D visualization platform for computational neuroscience research.

Updated Process Step 1

By fitting the data with the vision, action, and choice kernels defined in the referenced article, and employing half-Gaussian filters, we gain a comprehensive understanding of the data processing pipeline. This approach allows us to effectively assign each spike to a specific function by capturing the underlying patterns in neural activity. The use of these kernels ensures that the functional categories—vision, action, and choice—are represented with high precision, while the half-Gaussian filters enhance the sensitivity of the model, allowing for a more accurate mapping of spikes to their respective functions. This methodology not only deepens our insight into the neural processes at play but also ensures that each spike is correctly classified, providing a robust framework for functional analysis.

Updated Process Step 2

The left image shows the result using standard linear gradient descent, which demonstrates poor performance due to its slow and ineffective convergence. After applying the Adam optimizer, the convergence improved significantly, yielding the expected results. However, the learning curve still does not follow the desired exponential decay pattern. Moving forward, we plan to explore alternative optimization algorithms and different formulations to enhance the training process. Additionally, we aim to identify and remove potential outliers from the dataset to evaluate whether this improves convergence performance.

Upcoming Process Step 3 & 4

In Step 3, by mapping the spatial locations of 1,000 spikes onto an imported 3D mouse brain model, we can achieve a comprehensive 3D visualization of spike distribution, effectively enabling precise spike localization. In Step 4, building on the work completed in Step 1, we can assign each spike to its corresponding functionality. We will then visualize functional categories—such as vision, action, and choice—by representing each with a distinct color. This approach transforms the 3D brain model into a chromatic image, analogous to the original 2D visualization, enhancing our ability to interpret functional regions in three dimensions.
